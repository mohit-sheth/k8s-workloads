# Kubernetes Workloads

Kube-burner workloads for vanilla Kubernetes environments.

## Overview

This project uses [kube-burner](https://github.com/kube-burner/kube-burner) to run tests in Kubernetes environments.

## Testing locally

To run the tests locally, you can use the Makefile, [bats](https://github.com/bats-core/bats-core) is required to run the tests.

```bash
make test
```
